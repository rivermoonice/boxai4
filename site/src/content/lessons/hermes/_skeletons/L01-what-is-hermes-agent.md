## Suggested Pitfalls (for §4)

1. Confusing "AI agent" with "chatbot" — Hermes isn't a chat window, it executes tasks.
2. Skipping the install lesson because you "just want to read" — you have to run it to learn it.
3. Picking Hermes when you should be in a coding harness like Claude Code (or vice versa).
4. Believing Hermes' 200K-star count means it's the best tool for you — it means the *project* is popular.
5. Expecting an AI agent to remember your work between sessions without setup.

## Glossary Seeds

- **AI agent harness** — software that wraps an AI model so it can take actions (read files, run commands, send messages) on your behalf.
- **Model** — the AI itself (Claude, GPT, DeepSeek). The harness is the *body*; the model is the *brain*.
- **Loop** — when a harness keeps working on a task until a success check passes, instead of stopping at the first attempt.
- **MCP (Model Context Protocol)** — a standard that lets an AI harness talk to outside tools (calendar, GitHub, Notion). Covered in L05.
- **Slash command** — a short command (like `/goal`) you type into Hermes to trigger a built-in skill.
